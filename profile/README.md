## ğŸ› ï¸ Tech Stack

Our platform is built with modern, production-ready technologies:

## ğŸ¤ Contributing

We welcome contributions from developers of all skill levels!

### How to Contribute

1. **Find an Issue**
   - Browse [open issues](https://github.com/search?q=org%3Aalternatefutures+is%3Aissue+is%3Aopen&type=issues)
   - Look for `good first issue` labels
   - Check individual repo contribution guides

2. **Fork & Clone**
   - Fork the repository you want to contribute to
   - Clone it locally and create a feature branch

3. **Make Changes**
   - Follow the code style of the project
   - Add tests if applicable
   - Update documentation

4. **Submit PR**
   - Push your changes and create a Pull Request
   - Describe your changes clearly
   - Link related issues

### Contribution Guidelines

- Follow conventional commits (`feat:`, `fix:`, `docs:`, etc.)
- Write clear, concise commit messages
- Add tests for new features
- Update documentation as needed
- Be respectful and collaborative
